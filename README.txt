Привет!

Ключи для запуска:

-m - число столбцов
-n - число строк
вместе задают случайную генерацию доски m x n;

- file
укажите имя файла (предполагается, что лежит в корне) с начальным расположением.

Соответственно либо задаёте m, n, либо file

Файл должен быть *.txt, разделители столбцов - пробелы,
разделители строк - символы переноса строки.
Пример - файл table.txt

Спасибо за внимание!
